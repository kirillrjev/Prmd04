## Самостоятельная работа 1. Консольный калькулятор

## Тема

Разработка консольного приложения-калькулятора на C#.

## Цель

Освоить базовые принципы программирования на C#:

работа с условными конструкциями;

циклы;

обработка пользовательского ввода;

организация логики приложения.

## Задачи

Реализовать арифметические операции: сложение, вычитание, умножение, деление.

Организовать консольное взаимодействие с пользователем.

Обрабатывать ошибки и некорректный ввод.

Сделать возможность многократного использования калькулятора без перезапуска.

Освоить работу с Git и оформление README.md.

Контрольные вопросы

Какие типы данных используются для чисел в C#?

int, double, float, decimal, long и др.

Как организовать бесконечный цикл с возможностью выхода?

Использовать while(true) с условием выхода break или флагом.

Какие способы обработки исключений вы знаете?

try-catch, try-catch-finally, throw.

Как преобразовать строку в число с обработкой ошибок?

double.TryParse() или int.TryParse().

Какие принципы чистого кода применены?

Читаемость, небольшие методы, проверка входных данных, обработка ошибок.
